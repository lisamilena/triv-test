## Instalación

Todas las dependencias se gestionan a través del fichero package.json

`$ npm install`

`$ npm start`

## [Webpack](https://webpack.js.org/)

Empaquetado con webpack: [webpack](https://webpack.js.org/concepts/).

Run: `$ npm start`

Build: `$ npm run build`

Watch: `$ npm run watch`
